# 🏔️ QWERy - Travel Landing Page (Bootstrap 5)

This project is a visually striking travel landing page built using the **Bootstrap 5** framework. It focuses on clean typography, responsive grids, and modern web aesthetics.

### ✨ Key Features:
- **Responsive Layout:** Built with the Bootstrap grid system for all screen sizes.
- **Hero Section:** A bold and engaging introduction with high-quality imagery.
- **Navigation:** A sleek, functional header menu for easy browsing.
- **Content Sections:** Well-organized layouts for destinations and travel stories.
- **Modern UI:** Using custom fonts and Bootstrap components for a premium feel.

### 🛠️ Technologies Used:
- **HTML5:** For semantic and organized structure.
- **Bootstrap 5:** For rapid, responsive UI development.
- **CSS3:** For custom styling and fine-tuning the brand colors.
